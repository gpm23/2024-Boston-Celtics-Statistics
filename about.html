About

This project is dedicated to the Boston Celtics team 2023-2024 in light of their success, as of 5/16/24. The main goal of this project for me was to introduce the ideas of pre-processing and manipulating data sets from data scraping. All data was scraped from www.basketballreference.com, and parsed using the Beautiful Soup package. The data was analyzed, manipulated and visualized using plotly. This project is purely for fun and is a gateway for more extensive data scraping projects in the future!
